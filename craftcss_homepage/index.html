Zero-Config Beauty for
Semantic HTML

A minimal, class-light framework that makes your raw HTML look professional immediately. Clean. Responsive. Accessible. Flexible. Typographic. Semantic.

Lightweight

Under 15KB gzipped. We prioritize performance and only load the CSS you actually need.

Responsive

Mobile-first grid system that adapts fluidly to any screen size without complex media queries.

Dark Mode

First-class support for light and dark themes, automatically respecting user system preferences.

Write HTML. Get Design.

CRAFTS targets standard HTML tags. You don't need to learn a thousand utility classes just to style a paragraph or a button.

  • Semantic syntax
  • Accessible defaults
  • No build step required
