Build Beautiful Interfaces Faster

CRAFTS is a semantic-first CSS framework inspired by Notion, Halfmoon, and Tailwind — so your HTML looks great immediately and scales to full dashboards without the bloat.

Why Choose CRAFTS?

Design tokens, semantic defaults, and a utility escape hatch.

Fast & Light

Zero configuration by default with ≤15KB gzipped core bundle.

Responsive

Container + grid utilities ensure layouts never touch the edges.

Accessible

WCAG contrast compliance and prefers-reduced-motion baked in.

Composable

Mix semantic HTML defaults with utilities only when you need them.
