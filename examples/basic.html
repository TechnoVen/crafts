Build Beautiful Interfaces

CRAFTS is a modern CSS framework that combines clean design, responsive layouts, and developer experience. Perfect for creating Notion-like interfaces and complex web applications.

Why Choose CRAFTS?

Semantic HTML First

Works beautifully with pure HTML. No complex class names needed for basic styling. Perfect for rapid prototyping and content-heavy websites.

Mobile-First Responsive

Built from the ground up to work perfectly on all devices and screen sizes. Responsive grid system and flexible components.

Theme System

Light, dark, and auto modes with easy customization. Consistent design tokens and CSS variables for complete control.
